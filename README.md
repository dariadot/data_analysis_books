# data_analysis_books
Analysis of a book database pulled from kaggle. Focused on language categorization.

Analysis was made to pass Data Analysis and Visualisation course that is part of UAM's Cognitive Science bachelor program.
Link to the database: https://www.kaggle.com/datasets/jealousleopard/goodreadsbooks
Special thanks to the author: Soumik

Special thanks and regards to my professor Dawid Ratajczyk who supervised my work.

Repository consists of:
- csv database
- python code saved in a jupyter source file
- analysis report saved in a pdf file (in polish language)
